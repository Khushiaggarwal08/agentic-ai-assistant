# Agentic AI Assistant

Personal ChatGPT-style assistant built using agentic architecture.

## Architecture Flow

User Question  

→ Controller ([app.py](http://app.py))  

→ Router (category + risk + tone)  

→ Domain Brain (finance / general)  

→ Judge (quality validation)  

→ Rewrite loop (if needed)  

→ Final Answer  

## Features

- Agentic routing system

- Domain-specific reasoning

- Quality evaluation loop

- Automatic answer rewriting

- RAG-based customer support copilot

## Tech Stack

Python, Flask, NLP, LLM Concepts, RAG, GitHub
